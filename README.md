# MongoDB with replica set

[Official MongoDB Image](https://hub.docker.com/_/mongo) replica set initialization on the first run (and with healthcheck)

## Usage

Env vars to set `default`

* MONGO_RS `rs0`
* MONGO_AUTHDB `admin`
* MONGO_USER `root`
* MONGO_PASSWORD `iddqd`
